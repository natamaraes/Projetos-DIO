Tags de Título

As tags de título são usadas para estruturar hierarquicamente o conteúdo de uma página HTML. São 6 tags, sendo elas:

  • h1 - o conteúdo mais importante
  • h2
  • h3
  • h4
  • h5
  • h6 - o conteúdo menos relevante

Tags de Formatação de Texto

  • A tag p é utilizada para definir parágrafos de texto em HTML. Cada parágrafo é exibido como um bloco independente, com espaçamento antes e depois do conteúdo. Essa tag é fundamental para organizar o texto em seções legíveis, tornando a leitura mais fluida e bem estruturada.
  • A tag mark serve para destacar trechos importantes dentro de um texto, como se fosse uma marcação com um marcador amarelo. É útil para chamar a atenção do leitor para informações-chave, como termos importantes, datas ou conceitos destacados.
  • A tag small é usada para exibir o texto em tamanho reduzido em relação ao texto padrão da página. Geralmente é aplicada a notas de rodapé, observações ou avisos secundários, mantendo a hierarquia visual e semântica da informação.
  • A tag i coloca o texto em itálico, geralmente para dar ênfase leve, destacar palavras estrangeiras, nomes científicos ou termos técnicos.
  • A tag strong é usada para dar destaque semântico a um trecho do texto, exibindo-o em negrito. Além da aparência visual, essa tag indica aos mecanismos de leitura e busca que o conteúdo dentro dela é importante, o que melhora a acessibilidade e a compreensão do texto.
  • A tag u adiciona sublinhado ao texto, destacando-o visualmente. No entanto, deve ser usada com moderação, já que textos sublinhados podem ser confundidos com links. É mais adequada para materiais de estudo ou simulações visuais.
  • A tag sub exibe o texto em subscrito, ou seja, abaixo da linha base. É usada em expressões químicas (como H2O) ou matemáticas, quando se deseja indicar índices ou elementos inferiores.
  • A tag sup mostra o texto em sobrescrito, ou seja, acima da linha base. É frequentemente usada para expoentes matemáticos (como x2) ou notas de rodapé em documentos.
  • A tag blockquote é usada para indicar citações longas ou trechos destacados de outro autor. O navegador normalmente exibe esse conteúdo recuado, diferenciando-o visualmente do restante do texto. É útil em textos explicativos, artigos e documentações.

Tags de Listas

A tag a cria links clicáveis (hiperlinks) para outros sites, páginas ou arquivos. O atributo href define o destino do link, e target="_blank" faz com que ele abra em uma nova aba. Essa tag é fundamental para a navegação entre páginas e para a criação de referências externas dentro de um conteúdo web.

Tags Gerais

  • ol, ul e li

    • Essas três tags trabalham juntas para criar listas organizadas. A ol define uma lista ordenada (numerada), enquanto ul define uma lista não ordenada (com marcadores). Cada item dentro dessas listas é declarado com li. Elas são úteis para organizar etapas, tópicos ou coleções de informações de forma clara e visualmente estruturada.

  • hr

    • A tag hr insere uma linha horizontal que separa seções de conteúdo. É usada para criar divisões visuais entre tópicos, blocos de texto ou partes de uma página, contribuindo para a legibilidade e a organização do layout.
